# Endpoints to deployed API-backend

'https://hu-full-stack-assignments.onrender.com/api/persons' : 
 - GET: Displays all persons
 - POST: Adds a person to the endpoint

'https://hu-full-stack-assignments.onrender.com/api/persons/<id>' : 
 - GET: Displays the selected person with the <id>
 - POST: Deletes the selected person with the <id> 

'https://hu-full-stack-assignments.onrender.com/api/info' : 
 - Displays the amount of contacts in the list and the current timestamp(UTC)
